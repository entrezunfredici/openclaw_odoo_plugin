## Contexte

- Phase: cadrage
- Ce livrable sert de fiche synthèse “1 minute” du projet (cible + résultat attendu).

## Objectif

- Avoir une description courte, claire, partageable, qui cadre le projet dès le départ.

## Périmètre

**IN**

- Cible + résultat attendu

**OUT**

- Analyse détaillée du besoin (fait dans un autre livrable)
